# AGENTS.md

## Language Rules
- По умолчанию все создаваемые в этом репозитории документы, документация, описания, Markdown-файлы в `.planning/` и поясняющие тексты должны быть на русском языке.
- Если документ уже существует на английском и пользователь не просил сохранить английский, его нужно перезаписывать на русском языке, сохраняя структуру, смысл и полезные ссылки на файлы.
- Имена файлов, идентификаторы кода, названия API, системных сущностей, ключей конфигурации и literal-значения из кода не переводятся без явного запроса.
- Технические термины можно оставлять на английском только там, где это общепринято и улучшает точность, но основной текст, заголовки и списки должны быть на русском.
- Если пользователь явно просит другой язык для конкретного документа, приоритет у запроса пользователя.

## Documentation Style
- Пишите кратко, предметно и с привязкой к реальным файлам репозитория.
- Для ссылок на код и структуру используйте точные пути в обратных кавычках.
- Не добавляйте общие рассуждения без опоры на кодовую базу.
